H&H PPF brand website - Workbuddy edit package

Start here:
1. Open index.html in Workbuddy to edit the web page.
2. The page style is in styles.css.
3. All images and visual assets are in the assets folder.

Recommended workflow:
- Keep index.html, styles.css, and the assets folder together in the same folder.
- When replacing an image, put the new image in assets and update the image path in index.html.
- The current page is a static website, so it can be previewed directly in a browser.
